Met deze opdracht ga ik uitleggen wat sommige van de html taal doen. De html taal die ik ga uitleggen zijn h1, p, a, img, table en ul/ol/li

h1

h1 is een HTML heading tag.
HTML headings zijn titels of ondertitels die u op een webpagina wilt weergeven. Je heb intotaal 6 HTML headings h1 tot h6. h1 is gebruikt voor hoofd titels en h6 is gebruikt voor informatie die niet zo belangrijk is.

p

p is de HTML tag voor paragrafen.
een paragraaf begint altijd op een nieuwe regel en browsers voegen automatisch wat witruimte (een marge) toe voor en na een alinea.

a

a is de HTML attributes tag.
HTML attributes bieden aanvullende informatie over HTML-elementen.

img

img is de HTML tag voor images.
images kunnen het ontwerp en het uiterlijk van een webpagina verbeteren.

table

table is de HTML tag voor tabellen
Met HTML-tabellen kunnen webontwikkelaars gegevens rangschikken in rijen en kolommen.

ul/ol/li

ul, ol en li zijn HTML list tags.
ul definieert een unordered list.
ol definieert een ordered list.
li definieert een list item.

did zijn maar paar van de vele HTML tags.
